Comments

Comments are used to write notes to ourselves or future programmers. They are not considered code. We can write anything we want in comments. They often contain information about what our code does. We have to main types of comments they are:
//single line comment
/*
multiple
line
comments
*/

There is a third type of comment that can be used to create java docs. Later in the course we will cover these comments but they are much more extensive in what they can do. For now we will be only using these comments. As you learn more and more about objects and methods we will utilise comments to ensure that our code is more readable.
