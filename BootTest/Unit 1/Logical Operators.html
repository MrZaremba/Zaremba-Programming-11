Logical Operators

Logical operators are usually used with Relational Operators. Logical Operators are operations that compare booleans. Since all relational operators evaluate to booleans that is why they are often used together. The general format of logical operators are:
<boolean><operator><boolean>
The result always again evaluating to a boolean. There are many different logical operators but for the purpose of this course we will only need two. They are:

Operator Description
&& "and" true if both are true
|| "or" true if 1 or both are true

We can compare multiple values as long as every single item in the comparison evaluate into a boolean. Logical operators also evaluate to booleans. This mean when dealing with relation operators and logical operators the result will always either be true or false.
