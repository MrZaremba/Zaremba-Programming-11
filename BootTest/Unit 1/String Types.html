Strings

So far we have only dealt with data that has a single piece of data. Strings are different in that they are not considered to be primitive data types. They consist of multiple primitive data, which are of type character. Strings are a "string" of characters to form words, numbers, symbols or a anything we need them to be. They are our first type of data that is Iterable. What that means is that the String has index positions. Index positions point to a value at specific spot. Here is an example of index positions of a string:

Index Value
0 H
1 e
2 l
3 l
4 o

Indexing

Indexing in programming always starts off at 0. This means in the string "Hello" there are 5 positions labeled 0 through 4. We can use code to read what hte value at each index position is. With strings however, they are what we call Immutable. This means we cannot go in and change the value of any index position. WIth other Iterable data we can but not strings. The following code shows how we can grab values of index positions:
String name = "Patrick Stewart";
String lastName = name.substring(8,15);
System.out.println(lastName);
 The output of the following code is Stewart. The example above is called string slicing. We can also concatenate strings. This means combine 2 strings together. Example:
String firstName = "John";
String lastName = "Stewart";
String fullName = lastName + firstName;
 This will combine the strings together to get the a full name.
