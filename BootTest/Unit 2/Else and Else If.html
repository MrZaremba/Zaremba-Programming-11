Else and Else if Statements

Else statements

When we were creating if statements the block of code inside the statement would run only if the condition is true. After words regardless if the statement was true or false we could continue to run our programs. What if we want to run a block of code if the statement is true and a different block if the statement is false. This is where the else statement comes into play. Here is the general form of an else if statement:
if(condition){
 block of code if true
 }
else{
 block of code if false
 }

Notice that the else statement does not have a condition. There is no need for one as it is only going to run when its preceding if statement's condition is false.

Else if Statement

The else if statement is similiar to the else statement. The main difference is that it requires its own condition to execute its block of code. Here is the general format for the else if statement:
if(condition){
 block of code if statement is true
 }
else if(condition){
 block of code if statement is true
 }
else{
 block of code if all above statements are false
 }

There can be multiple else if statements after the initial if statement. Each one can only be acccessed if all the conditions prior to it evaluate to false. If any of the conditions through out the statement chain ever evaluate to true. The corresponding block of code will run and every successive statement will be skipped over.
