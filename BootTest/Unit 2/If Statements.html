If Statements

If statements are the first means of what we call Control Flow. Control flow of a program is the ability for the compiler to make decisions based on conditions. The general form of an "if" statement in java is:
if(condition) {
code;
}

Conditions will always evaluate to true or false. The way they work is if the condition is true, then we execute the block of code. If it is false then we skip over the block of code and continue with our program.

Branching Programs

Using the if statement we can create what is called a simple branching program. We can make checks and follow different branches further down our program. Here is sample code to create a basic branching program:
int age = 16;
int minDriverAge = 16;
if(age < minDriverAge){
  System.out.println("You are too young to drive");
  }
