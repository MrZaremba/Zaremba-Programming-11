While Loop

The problem with branching programs is that the code only flows in one direction. Take your paper rock scissors game for example. Once the user submitted their input the code ran the program ended. It would be better if the code would go back to the beginning and start over. It could keep doing this until the user puts in a certain input. That is one use for loops or iterations. Here is an example of the format of a while loop:
while(condition){
 block of code
 }

The condition just like in if and else if statements always evaluates to a boolean. The loop will continue to run as long as the condition is true. Once it turns false the loop will stop and the code will jump past the block.

Do While Loop

This is very similiar to the while loop. The only difference is that it checks the condition after the loop is complete. This should be used in a case where you want to guarantee the loop runs at least once. The general format for the do while loop is:
do{
 block of code
 }while(condition);
This will run the block of code then check the condition.

Infinite Loops

Infinite loops occur when the programmer does not lay in a condition that will every become false. This can occur by accident and will cause the program will crash or freeze your computer. Be sure to double check your conditions when using iterations.

Check Your Understanding

What will be the output of the following code:
int x = 5;
while(x > 0){
 System.out.print(x + ", ");
 x--;
 }

5, 4, 3, 2, 1
4, 3, 2, 1, 0
5, 4, 3, 2, 1, 0
The solution is 5, 4, 3, 2, 1
It starts out at 5 because the decrement to x is after the print statement. It does not print 0 because once 'x' is 0 the condition is false and the loop will not run.
