Arrays

An array is our second type of non-primitive form of data. An array is a piece of data that contains multiple peices of data. We can make an array of any type. Here is an example of instantiating an integer array.
int[] integers = new int[5];
 the square brackets lets the compiler know it is creating an array of integers and not just an integer. We have to put the number 5 in the square brackets later as we need to state what size the array is before using it. The size of an array cannot be changed, ever. Arrays similiar to strings but different are also indexed. We can insert the values into the array by setting an index value of the array to the primitive data:integers[2] = 10;.
