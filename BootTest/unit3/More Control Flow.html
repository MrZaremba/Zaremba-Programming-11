More on Control Flow

There are a few more items we can now add on to our control flow. We will learn how to use a "switch" statement and we will learn how to use a nested loop

Switch Statement

These statements operate similiar to "if" and "else if" statements. In fact switch statements could be done using if and else if statements. However, sometimes it is just easier to use a switch. Especially when dealing with a type of data called "enumerations" (These will not be covered). A switch statement takes in a variable and checks various cases on it. When it finds the appropriate case it will execute the block of code. Example:
String dayOfWeek = "Friday"; switch(dayOfWeek){  case "Monday":
  System.out.println("It is Monday");
  break;
 case "Tuesday":
  System.out.println("It is Tuesday");
  break;
 case "Wednesday":
  System.out.println("It is Wednesday");
  break;
 case "Thursday":
  System.out.println("It is Thursday");
  break;
 case "Friday":
  System.out.println("It is Friday");
  break;
 case "Saturday":
  System.out.println("It is Saturday");
  break;
 case "Sunday":
  System.out.println("It is Sunday");
  break;
 }
Once it finds the case equal to "Friday" it will execute that block of code. Notice the break statements on each case. Without these it would keep running.

Nested Loops

Nested loops are loops within loops. These can be very useful for different algorithms such as sorting or working with something we will get to later called 2d arrays. When running nested loops learning to follow the blocks of code can be tricky. Here is a simply example:
for(int i=1; i<11;i++){
 for(int j = 1;j<6;j++){
  System.out.print(i*j + "\t");
  }
 System.out.println("");
 }
Would create a multiplication table. Give it a try in Intelij to see what it looks like.
