Static Methods

Methods or functions are blocks of code that have been given a "name". They are like functions in math. They take in some input and they give an output. We have seen a few functions so such as Math.pow(2,3). Math is not a function which we will not refer to as methods. The pow(2,3) is the actual method. The Math portion is called a class which we will be getting to soon. Methods can take in parameters and return values. Or it could do neither. It depends what we want the method to accomplish. Here is the general format for a method:
signature name(parameters){
 block of code
  return value if there is one
 }

Naming methods should always be a good description of what the method accomplishes. This is another example of data abstraction. This method is essentially going to become a piece of data and we need to give it meaning as well. Naming conventions in java for methods are first word in name starts with a lowercase letter and any consecutive words start with an upper case letter. Note your method name should rarely be more than two words in length.

Signature

Signatures in Java a few parts to them. They are declaring a few things about method before we get to the name and parameters. The first part of the signature is accessibility, what classes are allowed to use this method. The second part of the signature is stating the return value. This will be a data type or void. If you have a data type the method must return that data type. If it is void then it returns nothing.

Parameters

Parameters are used to give the method additional information. For example the Math.pow(2,3) requires to parameters to run. It needs the base of the power and the exponent of the power. Without these how could it know what you wanted?

Return Values

Methods do not have to return anything. However, if they do return a value, that value will be sent to where the method is "called". Calling a method is how we invoke the code to run. For example, when we typed in Math.pow(2,3) that was us calling the method. When it returned its double value it would be inserted where the method was called. This would evaluate the invocation of the method to the return type. If a method has no return type then it simply would evaluate to null.

Example of a static method

Here we will show you an example of a static method that can calculate the pythagorean theorem of a right angle triangle
//returns the hypotenuse of a right angle triangle given side 'a' and 'b' static double pyth(double a, double b){
 return Math.sqrt((a*a + b*b));
 }
Here is how we could call it in our code:
double a = 4;
double b = 3;
double c = pyth(a, b);
This will set the double c to the value of the hypotenuse of the right angle triangle.
