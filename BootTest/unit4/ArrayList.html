Using the ArrayList Class

To start we need to import the class at the top of our project. The code to write this is import java.util.ArrayList; This gives us access to the class. We can import simply by creating a variable with type "ArrayList<object type>" then the IDE will underline it in red and ask if we want alt + enter and that will automatically input it. Here is an example of creating an arrayList to hold cars:
ArrayList<Cars> carList = new ArrayList<>();
The angled brackets dictating what type of object is being held in the array list are called generics. This is a topic beyond the scope of this course. We just simply need to know how to use it.

Using the class built in methods

There are several built in methods in the ArrayList class that we can utilize. Here is a list of the most common:

Method Functionality
add(int i, E element) Adds an item to list at position 'i'(or just use E element as parameter and added to end of list)
clear() Clears the list
get(int i) Gets the element stored at position 'i'
isEmpty() Returns true if list is empty, else it returns false
remove(int i) Removes element at index position 'i'
size() Returns an integer of size of list

There are others. All java libraries come with a Java Doc. You can find the ArrayList java doc by clicking here. This shows a list of all the methods and the parameters along with a description. There is more information on the class such as its constructors. I will let you explore it. Any built in class in java we are using will have a javadoc. To access them you can typically google the class name and javadoc to find it. Later in the course I will show you how to generate javadocs for your projects.

ArrayLists of Primitives

The array list cannot hold primitives it can only hold objects. Java has created several built in objects that are designed to hold primitives. So if wanted to make an ArratyList of integers we can use the Integer object.
ArrayList<Integer> myNums = new ArrayList<>();
 This will create a list holding Integer Objects. You have briefly seen this in use when we used the Integer.parseInt(String s) to change a string into an integer.
