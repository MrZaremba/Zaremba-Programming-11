Using Classes

Last lesson we looked at how to do basic design of a class. Now we will look at actually using a class in our program. The first to use a class is to instantiate it. This means we are going to make what we call an instance of the class. Each instance of a class will contian all the same fields and methods as any other instance of that class. However, the value held in memory at the fields can be different. For example with our car class all instances will have the variable called maxSpeed however the value held there can be different for each instance. To create an instance of a class we will use the new keyword. This tells the compiiler to create a spot in memory that holds the new class and all of its information. Example:
Car toyota = new Car();
 This code will create an instance of the car class and the variable toyota will point to its location in memory. We can now access all the public fields and methods inside the class. If we were to create another car instance we could set its fields to be different from the first. This is the main idea of what we call Object Orientated Programming. We create objects that represent any Type of data we want. This is also referred to as data abstraction

Constructors

Constructors are a special kind of method that runs when we instantiate an object. This constructor can be used to "setup" our object with desired parameters. Example of the constructor method inside our class:
Car(int maxSpeed, String colour, String name){
 this.maxSpeed = maxSpeed;
 this.colour = colour;
 this.name = name;
 currentSpeed = 0;
 }
This will run whenever we construct a car object. It allows us to set values of the fields when it is created. Notice the this keyword. It is referring to the object itself and when we set it equal to a value without this we are setting it equal to what ever is being passed in as a parameter.

The Accessor

You have used accessors several times already. They allow us to go into an object and access their public methods and fields. We use "." to access the properties of an object. Example:
toyota.speedUp();
This will go into the toyota instance of a class and run the method called speedUp. Sometimes objects can hold other objects. In this case we can go into the first object and then use the reference name for the second object to access its fields or methods. Example:
toyota.driver.getAge()
In this example the car class holds a reference to an object. Perhaps a person object. There reference is called driver. When we access car class we can access Person class through the driver reference inside of the car class and then run its method of getAge().
