Implementing

So far we have specified our data abstraction, determined the usage and created tests. Now we need to implement the code until all of our tests pass. We will go through method by method and code each one.

Insert Method

We will us an arrayList to hold our integers for first we need to instantiate an array list and also create a new arrayList in our constructor.

size method

Notice when we run our tests they still fail. We need to implement the size method now. The arrayList class already has a size method so we can just use that one. We can simply return size() of the list.

contains method

The tests still fail until we implement our final method. We can use the ArrayList method contains to check if our arrayList contains our integer. You might think that this is not good programming using all the built in methods from ArrayList, However, this is actually a huge part of programming using other classes that have already been made to write our own.
